# Example application

This folder contains a small Flutter application that demonstrates:

- package initialisation;
- online YouTube playback;
- controller commands;
- authorised offline download;
- local playback after a successful download.

## Run locally

From the repository root:

```bash
cd example
flutter create .
flutter pub get
flutter run
```

`flutter create .` generates the Android, iOS, macOS, Windows, and web runner folders for this example.

Before testing on Android, iOS, or macOS, apply the relevant platform configuration from the root [`README.md`](../README.md).
